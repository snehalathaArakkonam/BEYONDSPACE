PATTERN FOR EACH FILE:
```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>[PAGE_TITLE] • BEYONDSPACE</title>
    <link rel="stylesheet" href="css/main.css">
    <style>
        @import url('https://fonts.googleapis.com/css2?family=Orbitron:wght@400;700&family=Inter:wght@400;500;700&display=swap');
    </style>
</head>
<body>
    <canvas id="particles" class="particle-canvas"></canvas>
    
    [PASTE NAVBAR FROM HTML_COMPONENTS.html]
    
    <header class="page-header">
        <h1 data-en="[TITLE]" data-te="[TELUGU]">[TITLE]</h1>
        <p data-en="[DESC]" data-te="[TELUGU_DESC]">[DESC]</p>
    </header>
    
    <main class="[page]-main">
        [PAGE_SPECIFIC_CONTENT]
    </main>
    
    [PASTE FOOTER FROM HTML_COMPONENTS.html]
    [PASTE MODAL FROM HTML_COMPONENTS.html]
    [PASTE BACK-TO-TOP FROM HTML_COMPONENTS.html]
    
    <script src="js/particles.js"></script>
    <script src="js/main.js"></script>
    [PASTE PAGE-SPECIFIC SCRIPTS HERE]
</body>
</html>
```

IMPORTANT: Update the active nav-link for current page!
Change: <a href="page.html" class="nav-link"
To:      <a href="page.html" class="nav-link active"
