# 🚀 BEYONDSPACE – Interactive Space Tourism Platform

BEYONDSPACE is a futuristic, highly interactive multi-page **space tourism web platform** developed using **HTML5, SCSS, and Vanilla JavaScript**.  

It is designed to make space exploration **engaging, educational, and visually immersive**, especially for students, young learners, and space enthusiasts.

🔗 **Live Demo:** https://beyondspace27.netlify.app/

---

## ✨ Key Highlights

- 🌌 Modern **Dark Neon UI** with immersive space theme  
- 🚀 Fully interactive **multi-page architecture**  
- 🎮 Rich **JavaScript-driven features** (simulations, games, dynamic UI)  
- 📱 **Responsive design** (mobile, tablet, desktop)  
- 🎨 Advanced styling using **SCSS (modular, reusable, animations)**  
- ⚡ Smooth animations, transitions, and glowing effects  

---

## 🧩 Core Features

- 🌍 **Interactive Solar System** with real-time planet exploration  
- ⏳ **Mars Mission Countdown Timer**  
- 📰 Dynamic **Space News Ticker**  
- 🚀 **Rocket Launch Simulator** with physics-based controls  
- 🎮 Gamified learning with **Quiz System (multi-level + scoring)**  
- 🧑‍🚀 **Astronaut Avatar Generator** with downloadable ID  
- ✈️ **Trip Customizer** with dynamic itinerary generation  
- 🖼️ **Cosmic Gallery** with lightbox preview  
- 💬 **Feedback System** with validation & rating  
- 🌐 **Multilingual Support** (English + Telugu)  

---

## 🧠 Advanced Functionalities

- 🌌 Canvas-based **Particle Star Background**  
- 🎯 **Achievement System** using LocalStorage  
- 🚀 Interactive UI elements (parallax, hover effects, transitions)  
- 🎵 Optional **sound effects integration**  
- 🎮 Custom-built **game logic & simulations**  

---

## 🛠️ Tech Stack

- **Frontend:** HTML5, SCSS, Vanilla JavaScript  
- **APIs/Browser Features:** Canvas API, LocalStorage  
- **Design:** Responsive UI, Animations, Component-based structure
